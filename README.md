# Jackson but package in jmod 

## Dependencies
you just need `bash`, `core-utils`, `wget`

## Usage
Simply just:
```shell
git clone https://github.com/Mahisokhary/jackson-jmod
make
```
You can specify `VERSION`, `MAVEN_REPO` and `GROUP` like this:
```shell
make VERSION=2.18.0
```
